# [토일렛맵]

[01] 한 줄 피치 — 맛집·카페·바를 자주 찾는 30대 여성이 매장명만 검색하면, 방문 전 10초 안에 화장실 컨디션(청결도·위치·남녀공용 여부)을 확인할 수 있는 서비스

[02] 누가 쓰는가 — 월 4회 이상 맛집·카페·바를 약속 또는 혼자 방문하며, 화장실 컨디션이 별로면 당일 약속 경험을 "실패"로 인식하는 30대 여성

[03] 어떻게 쓰는가
- 입력 — 방문할 매장명 한 개 (텍스트, 최대 30자 / 한국어·영어·숫자)
- 처리 — 카카오맵 API로 매장 기본 정보를 조회 → LLM이 웹·리뷰를 크롤링해 화장실 정보를 추출 → 초안 작성(CALL 1) → 룰 기반 검수·보정(CALL 2)
- 출력 — 분류·위치·SNS·화장실 사진·청결도·화장실 위치·남녀공용 여부·칸 수를 정리한 화장실 정보 카드 (400자 이내, 카카오톡 공유용 복사 버튼 포함)
